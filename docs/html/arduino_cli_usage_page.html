Arduino CLI Usage

A guide on how to compile and upload the heatX project using Arduino CLI.

Quick Note for Arduino IDE Users:

If you prefer a graphical interface, the heatX project can be edited and managed in the Arduino IDE. However, the Arduino CLI is designed for advanced users who want to work without an editor or automate the process.

Getting Started with Arduino CLI:

  1. Download the heatX Project: Download the heatX project archive and extract its contents into:
    C:\Users\<YourUsername>\Documents\Arduino
  2. Open Terminal:
    • Press Win + R, type cmd, and press Enter.
    • Navigate to the project folder:
      cd "%USERPROFILE%\Documents\Arduino"

  3. Configure Arduino CLI: Initialize the Arduino CLI configuration:

    arduino-cli config init --overwrite

    Set the directory for platform data:

    arduino-cli config set directories.data "%USERPROFILE%\Documents\Arduino\platforms"

    Set the directory for temporary downloads:

    arduino-cli config set directories.downloads "%USERPROFILE%\Documents\Arduino\platforms\temp"

    Set the directory for your projects:

    arduino-cli config set directories.user "%USERPROFILE%\Documents\Arduino"

    Set the directory for built-in libraries:

    arduino-cli config set directories.builtin.libraries "%USERPROFILE%\Documents\Arduino\platforms\libraries"
  4. Verify the Configuration: Check the current configuration:
    arduino-cli config dump
  5. Update the Core Index: Update the list of available cores and platforms:
    arduino-cli core update-index

  6. List Connected Boards: Check if your ESP32-S3 board is connected:

    arduino-cli board list

    Example output:

    Port Protokoll Typ Platinenname FQBN Kern
    COM5 serial Serial Port (USB) Unbekannt

    Note the port (e.g., COM5) for the next steps.

  7. Compile the heatX Project: Compile the sketch and store the build artifacts in the build directory:
    arduino-cli compile heatX --clean --log-level warn --log --build-path "%USERPROFILE%\Documents\Arduino\heatX\build"
  8. Upload the Firmware: Upload the compiled firmware to the ESP32-S3:
    arduino-cli upload heatX -p COM5 --log-level warn --log --input-dir "%USERPROFILE%\Documents\Arduino\heatX\build"
  9. Monitor the Serial Output: Open the serial monitor to view the device's output:
    arduino-cli monitor heatX -p COM5 --config 115200
  10. Combine Upload and Monitor: For convenience, you can combine the upload and monitor commands:
    arduino-cli compile heatX --clean --log-level warn --log --build-path "%USERPROFILE%\Documents\Arduino\heatX\build" & arduino-cli upload heatX -p COM5 --log-level warn --log --input-dir "%USERPROFILE%\Documents\Arduino\heatX\build"
    • Press Ctrl + C to exit the monitor.

Notes:

  • Ensure your board is properly connected and drivers are installed.
  • Replace COM5 with the appropriate port for your device.
  • The Arduino CLI setup is a one-time process for your development environment.
