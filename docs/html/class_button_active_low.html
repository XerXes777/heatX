Detailed Description

Class to handle button input with active-low logic and debounce handling.

This class provides functionality to handle a button connected to a GPIO pin with active-low logic. It includes debouncing to prevent false triggering due to mechanical switch bouncing.

Example Usage

#include "gpio_hx.h"
ButtonActiveLow button(2, 50); // Initialize button on GPIO pin 2 with 50 ms debounce time
void setup() {
Serial.begin(9600);
}
void loop() {
button.update(); // Update button state
if (button.isPressed()) {
Serial.println("Button is pressed!");
} else {
Serial.println("Button is not pressed.");
}
delay(100); // Small delay to avoid flooding the Serial monitor
}
