extern

Array of material names derived from materialPresets.

This array is populated for use in an LCD item list, containing the names of all supported materials.

◆ materialPresets

MaterialPreset materialPresets[_MATERIAL_COUNT]
extern

Array of material presets for heating profiles.

Each entry contains a material name and its corresponding heating preset values.

Materials:

  • "PLA": Preset for PLA material.
  • "PETG": Preset for PETG material.
  • "ASA": Preset for ASA material.
  • "ABS": Preset for ABS material.
  • "PP": Preset for PP material.
  • "PA": Preset for PA material.
  • "PC": Preset for PC material.
  • "TPU": Preset for TPU material.
