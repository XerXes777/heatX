GPIO Pin Layout

Comprehensive GPIO pin layout for the heatX device.

Overview

This page describes the GPIO pin assignments for the heatX device, detailing the functionality of each pin.

Detailed GPIO Functions

Notes

  • Unused pins are labeled as "Free".
  • Strapping pins are reserved for boot and configuration purposes and should not be modified during runtime.
  • Ensure proper pull-up or pull-down resistors for specific GPIOs as required by the ESP32-S3 hardware specifications.
  • All input pins are configured with the internal pull-up resistors enabled.
  • All output pins are active-high and drive a low-side circuit using an IRLZ44N MOSFET.
