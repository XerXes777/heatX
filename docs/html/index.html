heatX Documentation

Introduction

heatX is a device that dries filament and maintains an optimal filament temperature to improve 3D print quality.

Features

  • Easy Operation: Simple control with a rotary encoder.
  • Real-Time Display: Constant temperature and humidity readings on an LCD screen.
  • Predefined Presets: Optimized temperature settings for popular filaments like PLA, PETG, ABS, and more.
  • Efficient PID Control: Highly responsive and energy-efficient heating system.
  • Flexible Integration: Potential-free start/stop contacts, compatible with 3.3V to 24V control systems.
  • Status Indicators: LEDs for heater, fan, and start/stop status.
  • Dual Control Modes: Operate in temperature-controlled or humidity-controlled modes.
