User Guide

A comprehensive guide to using the heatX device.

Overview

heatX is designed to improve your 3D printing experience by maintaining optimal filament conditions. This guide provides step-by-step instructions for using the device effectively.

Using heatX

  1. Powering On:
    • Connect the device to a power source.
    • The device will initialize and display the current temperature and humidity on the LCD screen.
  2. Selecting Filament Presets:
    • Use the rotary encoder to navigate through the preset options for popular filaments (e.g., PLA, PETG, ABS).
    • Press the encoder button to select the desired preset. The device will automatically adjust to the optimal temperature and humidity for the selected filament.
  3. Manual Adjustment:
    • If needed, you can manually adjust the temperature and humidity using the rotary encoder.
    • Hold down the encoder button to switch between temperature and humidity settings.
    • Rotate the encoder to increase or decrease the values.
  4. Starting and Stopping:
    • Press the "Start" button to activate the heater and fan.
    • Press the "Stop" button to deactivate the system.

Indicators

heatX uses LED indicators to provide real-time status updates:

  • Heating LED: Illuminates when the heater is active.
  • Fan LED: Illuminates when the fan is active.
  • Start/Stop LED: Indicates the system's overall status.

Maintenance

To ensure optimal performance:

  • Clean the fan and heater regularly to prevent dust buildup.
  • Check connections periodically to ensure all components are securely attached.
  • Replace the filament storage containerâ€™s desiccant packs as needed to maintain low humidity.

Troubleshooting

  • Device not powering on: Check the power connection and ensure the power source is active.
  • Temperature or humidity not adjusting: Ensure the sensor is connected and functioning properly.
  • LEDs not illuminating: Verify the wiring and check for any loose connections.
