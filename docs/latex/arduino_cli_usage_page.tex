\chapter{Arduino CLI Usage}
\hypertarget{arduino_cli_usage_page}{}\label{arduino_cli_usage_page}\index{Arduino CLI Usage@{Arduino CLI Usage}}
A guide on how to compile and upload the heatX project using Arduino CLI.\hypertarget{arduino_cli_usage_page_autotoc_md0}{}\doxysubsubsection{\texorpdfstring{Quick Note for Arduino IDE Users\+:}{Quick Note for Arduino IDE Users\+:}}\label{arduino_cli_usage_page_autotoc_md0}
If you prefer a graphical interface, the heatX project can be edited and managed in the Arduino IDE. However, the Arduino CLI is designed for advanced users who want to work without an editor or automate the process.\hypertarget{arduino_cli_usage_page_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{Getting Started with Arduino CLI\+:}{Getting Started with Arduino CLI\+:}}\label{arduino_cli_usage_page_autotoc_md1}
