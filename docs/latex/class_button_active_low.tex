\doxysubsection{Detailed Description}
Class to handle button input with active-\/low logic and debounce handling. 

This class provides functionality to handle a button connected to a GPIO pin with active-\/low logic. It includes debouncing to prevent false triggering due to mechanical switch bouncing.\hypertarget{class_button_active_low_autotoc_md20}{}\doxysubsubsubsection{\texorpdfstring{Example Usage}{Example Usage}}\label{class_button_active_low_autotoc_md20}
