\doxysubsection{Detailed Description}
Custom BME280 sensor class for efficient status polling. 

This class extends the {\ttfamily Adafruit\+\_\+\+BME280} library to allow direct access to the {\ttfamily read\+Register} function, enabling efficient polling of the {\ttfamily BME280\+\_\+\+REGISTER\+\_\+\+STATUS} register to detect when new sensor data is available.\hypertarget{class_custom_b_m_e280_autotoc_md30}{}\doxysubsubsubsection{\texorpdfstring{Use Case}{Use Case}}\label{class_custom_b_m_e280_autotoc_md30}
The {\ttfamily read\+Register} method allows high-\/frequency polling of the BME280 sensor\textquotesingle{}s status register (e.\+g., bit 3\+: Measuring) to determine precisely when new data becomes available. The polling interval is managed to balance responsiveness and system efficiency.
