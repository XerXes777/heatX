\doxysubsection{Detailed Description}
Class to manage GPIO state with off-\/delay functionality. 

This class allows you to control a GPIO pin, ensuring that it stays on for a specified duration (off-\/delay) before turning off when requested.\hypertarget{class_gpio_off_delay_autotoc_md20}{}\doxysubsubsubsection{\texorpdfstring{Example Usage}{Example Usage}}\label{class_gpio_off_delay_autotoc_md20}
