\doxysubsection{Detailed Description}
PID controller class for controlling systems based on feedback. 

This class implements a PID (Proportional-\/\+Integral-\/\+Derivative) controller with\+:
\begin{DoxyItemize}
\item Support for proportional on error or measurement.
\item Adjustable sample time, output limits, and tuning parameters.
\item Automatic or manual modes.
