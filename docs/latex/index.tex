\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro_sec}
heatX is a device that dries filament and maintains an optimal filament temperature to improve 3D print quality.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_features_sec}

\begin{DoxyItemize}
\item {\bfseries{Easy Operation}}\+: Simple control with a rotary encoder.
\item {\bfseries{Real-\/\+Time Display}}\+: Constant temperature and humidity readings on an LCD screen.
\item {\bfseries{Predefined Presets}}\+: Optimized temperature settings for popular filaments like PLA, PETG, ABS, and more.
\item {\bfseries{Efficient PID Control}}\+: Highly responsive and energy-\/efficient heating system.
\item {\bfseries{Flexible Integration}}\+: Potential-\/free start/stop contacts, compatible with 3.\+3V to 24V control systems.
\item {\bfseries{Status Indicators}}\+: LEDs for heater, fan, and start/stop status.
\item {\bfseries{Dual Control Modes}}\+: Operate in temperature-\/controlled or humidity-\/controlled modes.
