heatX is designed to improve your 3D printing experience by maintaining optimal filament conditions. This guide provides step-\/by-\/step instructions for using the device effectively.\hypertarget{user_guide_page_autotoc_md10}{}\doxysubsubsection{\texorpdfstring{Using heatX}{Using heatX}}\label{user_guide_page_autotoc_md10}

\begin{DoxyEnumerate}
\item {\bfseries{Powering On\+:}}
\begin{DoxyItemize}
\item Connect the device to a power source.
\item The device will initialize and display the current temperature and humidity on the LCD screen.
\end{DoxyItemize}
\item {\bfseries{Selecting Filament Presets\+:}}
\begin{DoxyItemize}
\item Use the rotary encoder to navigate through the preset options for popular filaments (e.\+g., PLA, PETG, ABS).
\item Press the encoder button to select the desired preset. The device will automatically adjust to the optimal temperature and humidity for the selected filament.
\end{DoxyItemize}
\item {\bfseries{Manual Adjustment\+:}}
\begin{DoxyItemize}
\item If needed, you can manually adjust the temperature and humidity using the rotary encoder.
\item Hold down the encoder button to switch between temperature and humidity settings.
\item Rotate the encoder to increase or decrease the values.
\end{DoxyItemize}
\item {\bfseries{Starting and Stopping\+:}}
\begin{DoxyItemize}
\item Press the "{}\+Start"{} button to activate the heater and fan.
\item Press the "{}\+Stop"{} button to deactivate the system.
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{user_guide_page_autotoc_md11}{}\doxysubsubsection{\texorpdfstring{Indicators}{Indicators}}\label{user_guide_page_autotoc_md11}

\item {\bfseries{Fan LED\+:}} Illuminates when the fan is active.
\item {\bfseries{Start/\+Stop LED\+:}} Indicates the system\textquotesingle{}s overall status.
